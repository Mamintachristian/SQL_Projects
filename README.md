# 📊 SQL Projects

This repository contains my SQL practice, projects, and data analysis work using relational databases. It serves as a continuous learning space where I explore SQL concepts from fundamentals to advanced analytics.

---

## 🎯 Purpose of This Repository

The goal of this repository is to build strong foundations in SQL and data analytics by working on structured datasets and solving real-world business-style problems.

This includes practicing:

- Relational database design
- Data querying and transformation
- Analytical thinking using SQL
- Problem-solving using business questions

---

## 🧠 Topics Covered

### 🟢 Fundamentals
- SELECT statements
- Filtering with WHERE
- Sorting and limiting results
- Basic SQL functions

### 🟡 Intermediate Concepts
- GROUP BY and HAVING
- Aggregate functions (SUM, AVG, COUNT, MIN, MAX)
- Joins (INNER, LEFT, etc.)
- Subqueries

### 🔴 Advanced Concepts
- Window functions (RANK, DENSE_RANK, ROW_NUMBER)
- Partitioning data (OVER PARTITION BY)
- Complex analytical queries
- Top-N per group problems
- Multi-step query structuring (CTEs / subqueries)
